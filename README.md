# Mission Link AI Assistant

A high-fidelity, next-generation AI chat platform.

## Features
*   **Intelligent Chat**: Powered by GPT-4o, offering context-aware responses.
*   **Secure Access**: Simple and secure login with email or nickname.
*   **Modern UI**: Premium, dark-themed interface inspired by modern AI tools.

## Tech Stack
*   **Framework**: Next.js 15+ (App Router)
*   **AI**: Vercel AI SDK, OpenAI GPT-4o
*   **Database**: Prisma with PostgreSQL
*   **Authentication**: NextAuth.js
*   **Styling**: Tailwind CSS, Framer Motion

## Getting Started

1.  Clone the repository
2.  Install dependencies: `npm install`
3.  Set up environment variables in `.env`
4.  Run the development server: `npm run dev`
